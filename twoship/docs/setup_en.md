# 2 Ship 2 Harkinian Multiworld Setup

## Requirements

- The Archipelago-enabled 64-bit 2Ship build
- A supported Majora's Mask ROM used to create `mm.o2r`
- An Archipelago room address, slot name, and optional password

## Connecting

1. Place `mm.o2r` beside the 2Ship executable or in its configured data folder.
2. Start 2Ship and open **Rando > Archipelago**.
3. Enter the room address, slot name, and password supplied by the WebHost.
4. Select **Connect**, then return to File Select.
5. Wait for **Start Archipelago** to appear and create a new save.

The client enables the native randomizer check categories selected by the seed.
An Archipelago save is associated with its server, slot, and seed and cannot be
opened while connected to a different multiworld session.
